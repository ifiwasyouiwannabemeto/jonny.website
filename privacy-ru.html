Контактная форма

Когда вы связываетесь со мной через контактную форму, передается следующая информация:

  • Имя (обязательно)
  • Адрес электронной почты (обязательно)
  • Номер телефона (необязательно)
  • Тип мероприятия (необязательно)
  • Желаемая дата (необязательно)
  • Сообщение (обязательно)

Обработка осуществляется через сервис Formspree (Formspree Inc., США). Ваши данные будут использоваться исключительно для обработки вашего запроса и не будут использоваться в рекламных целях.
Данные обычно удаляются не позднее чем через 6 месяцев, если нет юридических обязательств по их хранению.

Куки, скрипты и отслеживание

Этот веб-сайт не использует куки для рекламы, анализа или отслеживания. Используются только технически необходимые скрипты (например, для анимаций или функций форм), которые не хранят персональные данные.

Для отображения и работы веб-сайта загружаются библиотеки JavaScript и CSS через так называемые Content Delivery Networks (CDN). Это технически необходимые процессы, которые не собирают и не хранят персональные данные.

Внешние ссылки (Социальные сети)

Веб-сайт содержит ссылки на внешние платформы (TikTok, Instagram, YouTube, Spotify). При переходе по этим ссылкам применяются соответствующие политики конфиденциальности сторонних поставщиков. Я не имею влияния на их обработку данных.
